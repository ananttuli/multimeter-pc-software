NOTE: JRE/JDK 1.8 is  pre-requisite

1. Download and install the latest version of Maven.
2. Installation/download instructions can be found at https://maven.apache.org/install.html
3. Navigate to the root Folder of the PC software i.e. "Prototype" folder inside "Source Code"
4. You should see a file named "pom.xml".
5. Open command prompt and navigate to this directory which contains "pom.xml"
6. Type in command "mvn clean"
7. Type in command "mvn package"
8. Type in command "mvn install"
9. While in the same directory, type this command: java -jar target/Prototype-1.0-SNAPSHOT.jar
10. This should run the latest version of the jar file that was just built/generated by the command "mvn package"

11. Further use Launch4j to generate exe from jar
12. Used InnoDB to create .exe installer from exe file
